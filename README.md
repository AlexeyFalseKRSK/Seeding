# 🌱 Seeding

Кроссплатформенное настольное приложение на Python и PyQt5 для автоматической детекции и анализа посадочного материала на изображениях и в PDF-документах с использованием моделей компьютерного зрения.

## 📌 Описание проекта

**Seeding** — программное приложение для автоматизации анализа изображений, содержащих сеянцы и элементы посадочного материала. Система позволяет загружать изображения и PDF-документы, выполнять детекцию объектов, классификацию частей найденных сеянцев, визуализировать результаты обработки в графическом интерфейсе и формировать итоговые PDF-отчёты.

Проект ориентирован на решение прикладной задачи в области компьютерного зрения и реализует полный пользовательский сценарий анализа изображений: от загрузки исходных данных до получения итогового отчёта.

## ✨ Основные возможности

- загрузка изображений в распространённых графических форматах;
- открытие и обработка PDF-документов;
- автоматическая детекция сеянцев с использованием моделей YOLO;
- классификация частей найденных объектов;
- отображение результатов в виде дерева слоёв;
- визуальный просмотр найденных объектов и их частей;
- поворот изображений и отдельных объектов с перерасчётом координат;
- отображение статистической информации по результатам анализа;
- генерация PDF-отчёта.

## 🖥️ Системные требования

- Python 3.10 или выше;
- Windows, Linux или macOS;
- файлы обученных моделей в формате `.pt`.

## 📦 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/AlexeyFalseKRSK/Seeding.git
cd Seeding
```

### 2. Создание виртуального окружения


Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

Вариант 1. Через `requirements.txt`:

```bash
pip install -r seeding/requirements.txt
```

Вариант 2. Через `pyproject.toml`:

```bash
pip install -e .
```

## 🚀 Запуск приложения

Базовый запуск:

```bash
python -m seeding.main
```

Запуск с указанием модели детекции:

```bash
python -m seeding.main --weights path/to/model.pt
```

Запуск с указанием моделей детекции и классификации:

```bash
python -m seeding.main --weights path/to/detect_model.pt --classify-weights path/to/classify_model.pt
```

## 🔧 Переменные окружения

Для задания путей к моделям можно использовать переменные окружения.


Windows:

```bash
set YOLO_WEIGHTS_PATH=path/to/detect_model.pt
set YOLO_CLASSIFY_WEIGHTS_PATH=path/to/classify_model.pt
python -m seeding.main
```

## 📖 Основной сценарий использования

1. Открыть изображение или PDF-документ.
2. Выполнить детекцию сеянцев на текущем изображении или на всех страницах PDF.
3. Просмотреть найденные объекты в интерфейсе приложения.
4. Выбрать объект для анализа его составных частей.
5. Выполнить классификацию частей объекта.
6. При необходимости повернуть изображение или отдельный объект.
7. Сформировать итоговый PDF-отчёт.

## 🧩 Назначение основных модулей

### `seeding/main.py`

Точка входа в приложение. Отвечает за запуск программы, разбор аргументов командной строки и инициализацию графического интерфейса.

### `seeding/config.py`

Модуль конфигурации приложения. Содержит основные параметры и константы, используемые в системе.

### `seeding/controllers.py`

Модуль координации пользовательских сценариев. Связывает графический интерфейс с прикладной логикой и управляет основными действиями пользователя.

### `seeding/services.py`

Сервисный слой приложения. Реализует обработку изображений, запуск детекции, классификацию и подготовку данных для визуализации и отчётности.

### `seeding/inference.py`

Модуль выполнения инференса моделей компьютерного зрения. Содержит функции и логику запуска обученных моделей.

### `seeding/models.py`

Модуль структур данных приложения. Описывает сущности, используемые для хранения изображений, объектов, частей объектов и текущего состояния обработки.

### `seeding/report.py`

Модуль формирования PDF-отчётов. Отвечает за подготовку и сохранение результатов анализа в виде итогового документа.

### `seeding/ui/`

Пакет пользовательского интерфейса. Содержит главное окно приложения и вспомогательные графические компоненты.

### `seeding/utils/`

Набор вспомогательных функций общего назначения, используемых в различных частях проекта.

## 📚 Основные зависимости

Проект использует следующие библиотеки:

- `PyQt5` — построение графического интерфейса;
- `numpy` — численные вычисления;
- `opencv-python` — обработка изображений;
- `Pillow` — работа с растровыми изображениями;
- `PyMuPDF` — открытие и обработка PDF-документов;
- `ultralytics` — использование моделей YOLO;
- `reportlab` — формирование PDF-отчётов.

## ⚠️ Ограничения

- для работы приложения необходимы подготовленные модели в формате `.pt`;
- файлы весов моделей не включены в репозиторий;
- скорость обработки зависит от размера изображения и производительности оборудования;
- обработка многостраничных PDF-документов может занимать значительное время.
